# The config Tab

Every Component Manager Google Sheet has a **"config" tab** that stores the settings and connection details for that sheet. Most keys are managed automatically by the Component Manager; a few are meant to be set or changed by you.

{% hint style="warning" %}
Only change keys marked **Configurable** in the table below. Editing **Fixed** (system-managed) keys can break the connection to Adobe Analytics or the add-on's internal state.
{% endhint %}

The **Type** column tells you whether a key is:

* **Configurable** â€” you may set or change it. Some configurable keys (the credentials) are filled in for you during the [initial setup](../getting-started-with-the-component-manager/) and normally only change when you renew or migrate your Adobe API connection.
* **Fixed** â€” written and maintained by the Component Manager automatically. Don't edit these by hand.

{% hint style="info" %}
The example values below are **anonymized placeholders**, not real credentials. Your own values will differ.
{% endhint %}
